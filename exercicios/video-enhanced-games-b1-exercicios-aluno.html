Exercício 4 de 5

Grammar in use — expressing approval and disapproval

Study the expressions below. Then complete each sentence using the structure indicated, sharing your real opinion about the Enhanced Games.

Expressing approval and disapproval in Portuguese

Aprovação

  • Apoio essa iniciativa.
  • Sou a favor de + substantivo
  • Concordo com a decisão de + infinitivo
  • Faz todo sentido que + subjuntivo
  • É válido / louvável que + subjuntivo
  • Me parece uma ideia acertada.
  • O que me agrada é que + indicativo

Desaprovação

  • Sou contra essa medida.
  • Não apoio / discordo de + substantivo
  • Não faz o menor sentido que + subjuntivo
  • É questionável / preocupante que + subjuntivo
  • O que me preocupa é que + indicativo
  • Isso normaliza / banaliza…

1. [É preocupante / É louvável que…]

2. [Concordo com / Discordo de…]

3. [O que me preocupa é que… / O que me agrada é que…]

4. [Isso normaliza… / Isso demonstra…]
